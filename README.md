🛒 App de Catálogo de Produtos - Flutter
Este projeto é um aplicativo de gerenciamento e visualização de produtos interativo, desenvolvido para a disciplina de Desenvolvimento para Dispositivos Móveis da Faculdade Senac Joinville.

Estudante: João Vitor Prestes Garcia
Turma: 5ª Fase ADS 2026

🚀 Funcionalidades Implementadas
Listagem de Produtos: Exibição dinâmica de itens com imagem, título e preço.
Visualização de Detalhes: Tela específica para exibir informações detalhadas e descrição de cada produto.
Gerenciamento de Favoritos: Funcionalidade para marcar/desmarcar produtos como favoritos com atualização em tempo real.
Carrinho de Compras: Sistema para adicionar itens, controlar quantidades e calcular o valor total automaticamente.
Notificações (Snackbars): Feedback visual ao adicionar itens ao carrinho, permitindo desfazer a ação.
Interface Moderna: Design responsivo utilizando componentes do Material 3 e navegação por abas/gaveta (Drawer).

🛠️ Requisitos Técnicos
Arquitetura de Modelos: Implementação de classes de modelo para Produto e Carrinho.
Gerenciamento de Estado: Uso do pacote Provider para comunicação eficiente entre componentes e gerenciamento global de dados.
Navegação Avançada: Utilização de rotas nomeadas para transição entre telas.
Comunicação Reativa: Atualização automática da UI baseada em mudanças no estado dos dados.

🏃 Como Executar
Certifique-se de ter o Flutter instalado em sua máquina.

Clone este repositório:
git clone https://github.com/JoaoGarcia07/app-produtos-flutter-joao.git

Acesse a pasta do projeto e execute:
flutter pub get

Inicie o aplicativo em um emulador ou dispositivo físico:
flutter run
